# CRM Management System

A simple **Customer Relationship Management (CRM) system** built using **PHP, MySQL, HTML, CSS, and TailwindCSS**.
The system allows administrators to manage **customers, leads, and products** from a centralized dashboard.

This project demonstrates **CRUD operations, database integration, authentication, and admin panel management**.

### Admin Authentication

### Customers Management

- View all customers in **table format**

- **Delete customers**

### Leads Management

- View all leads
- Search leads by **name**
- **Add new leads**
- Edit existing leads
- Delete leads
- View lead details

### Product Management

- View all products
- Search products by **name**
- **Add new products**
- Edit product information
- Delete products
- View product details
